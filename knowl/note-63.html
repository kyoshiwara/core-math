Note 4.1.17.

The sum in Example 4.1.16, \(\dfrac{13}{12}\text{,}\) is an improper fraction because its numerator is larger than its denominator. That just means that the fraction represents a number bigger than one whole. In this case, because \(\dfrac{12}{12} = 1\text{,}\) we can write
\begin{equation*} \dfrac{13}{12} = \dfrac{12 + 1}{12} = \dfrac{12}{12} + \dfrac{1}{12} = 1 + \dfrac{1}{12} \end{equation*}
We usually write \(1 + \dfrac{1}{12}\) as a mixed number, \(1 \dfrac{1}{12}\text{.}\)
