Solution 7.2.3.1.

  1. Subtracting 10 gives the same result as adding \(-10\text{,}\) so we can write
    \begin{equation*} 6 - (+10) = 6 +(-10) \end{equation*}
    To add two numbers with opposite signs we subtract their unsigned parts to get \(10 - 6 = 4\text{.}\) And because 10 is greater than 6, the answer is negative. Thus
    \begin{equation*} 6 - (+10) = 6 +(-10) = -4 \end{equation*}
  2. Subtracting 3 gives the same result as adding \(-3\text{,}\) so we can write
    \begin{equation*} -4 - (+3) = -4 +(-3) \end{equation*}
    To add two numbers with the same sign we add their unsigned parts to get \(4 + 3 = 7\text{.}\) And because both numbers are negative, the answer is negative. Thus
    \begin{equation*} -4 - (+3) = -4 +(-3) = -7 \end{equation*}
