Note 6.3.11.

In the Example above, note that \(C\) is not proportional to \(t\text{;}\) the graph does not pass through the point \((0,0)\text{.}\) At \(t = 0, ~C = 5\text{.}\) (Perhaps there is a $5 deposit on the bike.) Nonetheless, we can compute the slope of the line by calculating the change in its variables as we move from one point to another.
