Example 5.1.9.

Combine like terms.
  1. We add the coefficients, \(4 + 3\text{,}\) and leave the variable part, \(x\text{,}\) unchanged.
    \begin{equation*} 4x^2 + 3x^2 = (4 + 3) x^2 = 7x^2 \end{equation*}
  2. \(7a^2\) and \(5a\) are not like terms, because the variable, \(a\text{,}\) does not have the same exponent. So we cannot combine the terms; this expression cannot be simplified.
