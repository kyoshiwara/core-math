A Quick Refresher.

We have solved equations by undoing the operation performed on the variable. For example, in the equation
\begin{equation*} 4x = 28 \end{equation*}
the variable \(x\) is multiplied by 4, so we divide both sides of the equation by 4:
\begin{align*} \dfrac{4x}{\blert{4}} \amp = \dfrac{28}{\blert{4}} \hphantom {0000} \blert{\text{Divide both sides by 4.}}\\ x \amp = 7 \end{align*}
The solution is 7. We can check the solution by substituting \(\alert{7}\) for in the equation.
\begin{equation*} \text{Check:}~~~~4(\alert{7}) = 28~~~~ \blert{\text{True.}} \end{equation*}
Because this statement is true, 7 is indeed the solution.
